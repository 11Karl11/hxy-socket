#### 介绍
    基于springboot和netty的socket通信框架,方便扩展springboot各类插件。
    同时，框架本身
    1.支持websocket,tcp-socket连接
    2.json和protocolbuffer编码协议，以及其他自定义扩展协议
    3.超高性能和超简易的特性
